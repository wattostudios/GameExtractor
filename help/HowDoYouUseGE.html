How Do You Use Game Extractor

Back to Index

Open an archive from a game?

Click the Open Archive button at the top of the screen - the Side Panel changes to show a list of files and folders.
Locate the file you want to open, and double-click it.
Game Extractor will look at the archive, and determine if a plugin can open it. If so, the archive will be opened, and the File List will be filled with information.


This is the button to display the Open Archive Side Panel


Make changes to an archive?

At the bottom of the Side Panel, click the Modify Archive button, or click the Add Files, Remove Files, or Replace Files buttons at the top of the screen.
If the archive you opened can be edited, the controls will change to show operations like Add, Remove, and Replace files. You can use any of these buttons to modify the archive.
If you want to replace a file, the easiest way is to drag a file from the Side Panel onto the File List. When you let go of the mouse, the file under the mouse cursor will be replaced with the file you dragged. The size of the file will change, and the icon will change to show that the file has been replaced.


This the the button to open the Modify Archive functions

Save an archive, and use it in the game?

Click the Save Archive button at the top of the screen, or at the bottom of the Side Panel.
If the archive you opened can be saved, the controls will change to show the saving options.
Enter the name of your new archive, and click the Save button.
A progress bar is displayed to indicate the length of time left before the archive will be fully saved. The archive may take a while to save, depending of the number of files, and whether compression is used.


This is the button to display the Save Archive Side Panel

This function can only be used on archives that support saving.
